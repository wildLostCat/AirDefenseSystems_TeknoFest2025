# Overview:
This repo contains all of the code used by our team Hangar in the Air Defense Systems competition TeknoFest2025.

---
### Pin Layout:

**X stepper:**
```
direction pin -> 32
step pin -> 23
MS1 pin -> 13
MS2 pin -> 12
MS3 pin -> 14
```

**Y stepper:**
```
direction pin -> 33
step pin -> 25
MS1 pin -> 22
MS2 pin -> 16
MS3 pin -> 17
```
**Laser:**
`enable pin -> 18`
